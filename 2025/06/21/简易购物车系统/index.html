简易购物车系统

项目简介

本项目使用 Java 与 mysql 数据库建立一个简易购物车系统，模拟用户网购时使用购物车的操作，支持用户把特定商品加入购物车，并将商品进行结算。

项目采用技术

开发语言：Java 17

IDE: IntelliJ IDEA

UI 框架：Java Swing

数据库：mysql

辅助工具：Maven（打包）
